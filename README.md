# 해커톤 - 패스트캠퍼스 FE개발 16기
크롬 확장기능인 모멘텀을 클론하는 1일

## 최소구현기능
- ✔ 시간표시 <-> 일과 경과시간 퍼센티지 표시
- ✔ 시간에 따른 인사말 / 유저 이름 입력 / localStorage 관리
- ✔ 날씨정보 API 활용 - 예보 표시 / 사용자 위치에 따라 비동기 처리 후 렌더링
- ✔ 배경 이미지 API - 배경 정보 표시 및 1일 간 유지 / 고정 기능
- ✔ 투두리스트 기능 - 사용자 입출력 기록 및 localStorage 관리

## 개발계획
- ES6 모듈 기능을 활용한 파일 분산화
- git flow 정책으로 기능별 브랜치 생성/ 풀리퀘스트 방식
- prefix, layout 관련 컨벤션 합의
